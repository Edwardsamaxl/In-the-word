# 字里行间

《字里行间》是一款发生在阅读页面中的短篇文字探索游戏。

当前仓库实现了第一关《静夜思》的前端骨架，包括固定字符网格、移动、光、霜、子弹时间、撞月、下沉和第二关接缝。

## 项目类型

这是一个**纯前端项目**，当前没有后端、数据库或登录服务。

技术栈：

- Vite
- Vanilla JavaScript
- GSAP
- Canvas
- 原生 CSS

## 本地启动

需要 Node.js 20 或更高版本。

首次运行先安装依赖：

```bash
npm install
```

然后启动开发服务器：

```bash
npm run dev
```

终端会显示访问地址，通常是：

```text
http://localhost:5173
```

所以和常见的前后端项目一样，日常开发直接使用：

```bash
npm run dev
```

如果依赖已经安装过，不需要重复执行 `npm install`。

## 生产构建

```bash
npm run build
```

构建结果输出到：

```text
dist/
```

`dist` 中是可以部署到静态网站服务的纯前端文件，例如：

- Vercel
- Netlify
- GitHub Pages
- Nginx
- 对象存储静态网站

## 本地预览构建结果

先执行：

```bash
npm run build
```

再执行：

```bash
npm run preview
```

## 操作方式

### 键盘

| 操作 | 按键 |
|---|---|
| 向左移动 | `←` 或 `A` |
| 向右移动 | `→` 或 `D` |
| 跳跃 | `↑`、`W` 或空格 |
| 下沉 | 在“低”字上按住 `↓` 或 `S` 1 秒 |
| 重置 | `R` |

### 触屏

- 按住屏幕左半边：向左移动
- 按住屏幕右半边：向右移动
- 向上滑动：跳跃
- 在“低”字上下滑并保持 1 秒：下沉
- 点击右上角循环箭头：重置

## 第一关流程

1. 向右经过“光”，触发月光。
2. 继续经过“霜”，短暂冻结。
3. 自动换到第二行，经过“举”进入子弹时间。
4. 在子弹时间内跳跃，撞碎第一行并生成月亮。
5. 走到“低”字。
6. 按住下沉操作 1 秒，进入第二关接缝画面。

## 可用命令

| 命令 | 用途 |
|---|---|
| `npm run dev` | 启动开发服务器 |
| `npm run build` | 生成生产构建 |
| `npm run preview` | 本地预览生产构建 |
| `npm run smoke` | 使用本机 Chrome 自动完成第一关测试 |

`npm run smoke` 需要满足：

- 开发服务器已经运行
- 页面地址为 `http://127.0.0.1:5173`
- Windows 安装了 Google Chrome
